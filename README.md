# rusyle-gaming-platform
RuSyle — инновационная платформа для развития и популяризации российской игровой индустрии.  Приложение объединяет геймеров и разработчиков:  Для игроков: анонсы новых проектов, календарь событий и информация о грядущих обновлениях  Для разработчиков: инструменты для анонсирования и продвижения своих игр
